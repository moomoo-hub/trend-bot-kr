# 트렌드봇 (TrendBot) v1.0 — 봇 명세서

> 네이버 뉴스 기반 자동 UGC 콘텐츠 생성 및 텔레그램 발송 봇

---

## 개요

트렌드봇은 네이버 뉴스에서 실시간 트렌드 키워드를 수집하고,
Claude AI로 2030 직장인 커뮤니티 톤의 게시글을 자동 생성해
텔레그램으로 발송하는 자동화 봇입니다.

- 실행 주기: 2시간 간격, 하루 최대 7회 (09:00 ~ 21:59)
- 회당 발송: 최소 1개 이상 게시글 (3개 목표, 안정성 우선)
  - 1개 이상: 최소 발송 보장 (적시성 중시)
  - 3개 목표: 정상 상황에서 3개 모두 발송
- 중복 방지: 24시간 내 사용된 키워드 재사용 안 함

## 핵심 플로우

```
[1] 수집 (collector.py)
    ├─ RSS 피드 시도 (네이버 뉴스 분야별)
    │   └─ 실패 시 웹 크롤링으로 자동 전환
    ├─ 기사 본문 개별 추출 (URL 방문, 최대 500자)
    │   └─ 추출 실패 시 기사 제목으로 대체
    └─ 정치 기사 2단계 필터링
        ├─ [1단계] URL 패턴 (sid1=100) 제외
        └─ [2단계] 제목 키워드 블랙리스트 제외

[2] 선별 (selector.py)
    ├─ 24시간 내 사용된 키워드 제외
    ├─ 분야별 분류 (경제/사회/IT/연예/스포츠/생활문화)
    └─ 상위 3개 선택

[3] 생성 (writer.py)
    ├─ Claude AI로 UGC 스타일 게시글 작성
    │   ├─ 제목: 30~45자, 질문형 ("?", "거임?" 등)
    │   ├─ 본문: 350~450자, 3문단 구조
    │   │   ├─ 1문단: 뉴스 핵심 + 공감 ("나만 그런 거임?")
    │   │   ├─ 2문단: LSI 키워드 3개 이상 + 개인 경험담
    │   │   └─ 3문단: A vs B 질문 + 미래 예측
    │   └─ 음슴체 톤: "~임", "~함", "~더라", "~었음" 포함
    ├─ SEO 점수 80점 미만 또는 길이 이탈 시 최대 4회 재시도 (총 5회)
    └─ 5회 소진 시 검증 통과 가능한 Fallback 게시글 자동 생성

[4] 검증 (sender.py)
    ├─ 정치 키워드 포함 여부
    ├─ 최소 포스트 수 (1개 이상) ← v1.1에서 3개→1개로 완화 (안정성↑)
    ├─ 제목 글자 수 (30~45자)
    ├─ 본문 글자 수 (350~450자)
    ├─ 음슴체 마커 존재 여부 (Fallback 제외)
    ├─ 기사 링크 유효성 (Fallback 제외)
    └─ 라벨 문자열 금지 ("제목:", "본문:")

[5] 발송 (sender.py → Telegram Bot API)
    ├─ 포맷: 키워드[분야] + 제목 + 본문 + 기사링크
    └─ 검증 실패 시 전체 발송 중단

[6] 기록 (selector.py)
    ├─ 발송 성공 여부와 무관하게 사용된 키워드 기록
    └─ 저장 시 24시간 만료 항목 자동 정리
```

## 에러 처리

| 단계 | 에러 | 처리 방식 |
|------|------|-----------|
| 수집 | RSS 실패 | 웹 크롤링 자동 전환 |
| 수집 | 본문 추출 실패 | 기사 제목으로 대체 |
| 수집 | 기사 부족 (3개 미만) | 수집 결과 내 추가 보충 후 더미 Fallback |
| 생성 | 파싱 실패 / API 오류 | 최대 4회 재시도 후 Fallback 게시글 |
| 발송 | 검증 실패 | 전체 발송 중단 (키워드는 기록됨) |
| 발송 | API 오류 | 최대 3회 재시도 후 텔레그램으로 에러 리포트 |
| 전체 | 운영시간 외 실행 | 실행 스킵 (09:00~21:59만 운영) |

## SEO 점수 기준 (100점 만점)

| 항목 | 점수 | 조건 |
|------|------|------|
| 제목 길이 | +25 | 30~45자 |
| 본문 길이 | +25 | 350~450자 |
| 제목 질문형 | +15 | "?", "어?", "나만", "근데", "왜" 포함 |
| 키워드 포함 | +15 | 제목 + 본문 모두, 2회 이하 |
| LSI 키워드 | +15 | 2문단에 3개 이상 |
| UGC 톤 | +5 | "함", "임", "ㅋ", "ㅠ", "어?" 등 3개 이상 |

**발송 기준: 80점 이상**

## 실행 방법

```bash
# 즉시 1회 실행 (테스트용)
python main.py once

# 스케줄러 모드 시작 (2시간 간격, 자동 반복)
python main.py run

# 도움말 조회
python main.py help
```

**용도별 실행:**
- 개발/테스트: `python main.py once` (로그로 결과 확인)
- 정기 운영: `python main.py run` (백그라운드 실행 권장)
- 크론/스케줄러: `python /path/to/main.py once` (시스템 task scheduler 설정)
